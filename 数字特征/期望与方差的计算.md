## 第一部分：计算公式（离散型）

### 1. 期望 (Expectation)

对于离散型随机变量 $X$，假设其可能的取值为 $x_1, x_2, \dots, x_n$，对应的概率质量函数为 $P(X = x_i) = p_i$。

- **基本定义公式**：
    
    $$E(X) = \sum_{i} x_i p_i$$
    
- **随机变量函数的期望**（无意识统计学家法则，LOTUS）：
    
    若有一个新变量 $Y = g(X)$，其期望为：
    
    $$E(g(X)) = \sum_{i} g(x_i) p_i$$
    

### 2. 方差 (Variance)

方差衡量离散型随机变量取值在其期望值附近的偏离程度。

- **定义公式**：
    
    $$Var(X) = E[(X - E(X))^2] = \sum_{i} (x_i - E(X))^2 p_i$$
    
- **简便计算公式**（推导与应用中最常用）：
    
    $$Var(X) = E(X^2) - [E(X)]^2$$
    
    _(注：其中 $E(X^2) = \sum_{i} x_i^2 p_i$)_

## 第二部分：计算性质

### 1. 期望的计算性质

期望的核心在于其**线性（Linearity）**。假设 $X, Y$ 为离散型随机变量，$a, b, c$ 为常数。

- **常数的期望**：
    
    $$E(c) = c$$
    
- **线性变换**：
    
    $$E(aX + b) = aE(X) + b$$
    
- **和的期望**（无论 $X$ 与 $Y$ 是否独立均成立）：
    
    $$E(X + Y) = E(X) + E(Y)$$
    
- **乘积的期望**（**必须满足** $X$ 与 $Y$ 相互独立）：
    
    $$E(XY) = E(X)E(Y)$$
    

### 2. 方差的计算性质

方差对常数和系数的处理遵循“平方”法则。假设 $X, Y$ 为离散型随机变量，$a, b, c$ 为常数。

- **常数的方差**（常数没有波动）：
    
    $$Var(c) = 0$$
    
- **平移不变性**：
    
    $$Var(X + c) = Var(X)$$
    
- **缩放性质**（系数提出去必须平方）：
    
    $$Var(aX) = a^2 Var(X)$$
    
- **线性组合的方差**（一般情况）：
    
    $$Var(aX \pm bY) = a^2 Var(X) + b^2 Var(Y) \pm 2ab \cdot Cov(X, Y)$$
    
    _(注：$Cov(X, Y)$ 为 $X$ 和 $Y$ 的协方差)_
    
- **独立变量的方差**（**必须满足** $X$ 与 $Y$ 相互独立，此时 $Cov(X,Y)=0$）：
    
    $$Var(X + Y) = Var(X) + Var(Y)$$
    
    $$Var(X - Y) = Var(X) + Var(Y)$$
    
    _(注：只要变量独立，不管是加还是减，整体的方差（不确定性）都是叠加相加的)_
